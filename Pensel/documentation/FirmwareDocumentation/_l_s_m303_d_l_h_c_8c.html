Gets data from the accel queue.

Note: This is function modifies the tail index if we're overwritting unread data. LSM303DLHC_accel_putPacket also can modify it, so we must check for that case in this function as LSM303DLHC_accel_getPacket gets called asynchronously.

Parameters
pkt_ptr(accel_packet_t *): Place to put the retreived accel packet.
peak(bool): Whether or not we should pop it off the queue or to just peak at it.
Return values
(ret_t)Success or failure of getting the accel packet off the queue.

◆ LSM303DLHC_drdy_ISR()

void LSM303DLHC_drdy_ISR ( void  )

Interrupt handler for the DRDY interrupt pin coming from this chip.

  1. Check the status register of the chip, update error flags, and check for available data.
  2. Based on the status, get the available data.
  3. Put that data into a packet queue.
