◆ HAL_IncTick()

void HAL_IncTick ( void  )

millisecond ISR that increments the global time as well as calls some functions that need to be periodically serviced. We do a few things:

  1. Take care of button and switch debouncing every 10 ms
    • If we've changed states in interrupt context and enough time has elapsed (``) we will change the button / switch state
  2. Pet the watchdog every 5 ms. Must be pet within 10 ms!
  3. Toggle the heartbeat LED every 1 second
